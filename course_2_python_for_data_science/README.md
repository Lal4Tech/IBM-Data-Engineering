- ```type(21.213)```: ```float```

- ```type(True)```: ```bool```

### Expressions and Variables

**Expressions**:

- ```3 + 4 + 7```: ```14``` (here ```+``` is operator and numbers are operands)
- ```5 - 6```: ```-1```
- ```5 * 6```: ```30```
- ```25 / 5```: ```5.0```
- ```25 / 6```: ```4.166..```
- ```25 // 6```: ```4``` (Integer division)
- ```2 * 60 + 30```: ```150```
- ```30 + 2 * 60```: ```150```
- ```(30 + 2) * 60```: ```1920```

**Variables**:

- ```my_variable = 1```; my_variable:1
- ```my_variable = 10``` (now the value of the variable is changed to 10)
- ```x = 40 + 100 + 50```; x:150
- ```y = x / 10```; y:15.0
- ```x = x / 15```; x:10.0
